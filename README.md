Są to m.in.:
- Metoda stycznych (JAVA i Python)
- Metoda siecznych
- Metoda Falsi
- Metoda bisekcji
- Interpolacja Lagrange'a
- Schemat Hornera
- Wzór trapezów prosty i złożony
- Metoda Simsona prosta i złożona
